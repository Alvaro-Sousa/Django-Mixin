# 📚 Exercício Django-Mixin

Bem-vindo ao projeto de exercício! Este repositório contém exercícios práticos em Python usando o framework Django. 🚀

## 📝 Descrição do Projeto
Este projeto foi desenvolvido para ajudar você a aprender e praticar suas habilidades em Python usando Django. Ele inclui vários exercícios que demonstram os conceitos principais do framework.

## ✨ Conceitos Principais
- 🔹 Fundamentos do Django
- 🔹 Arquitetura MVC (Model-View-Controller)
- 🔹 Roteamento de URLs
- 🔹 Modelos de banco de dados e ORM
- 🔹 Templates e arquivos estáticos
- 🔹 Tratamento de formulários e validação
- 🔹 Autenticação de usuários

## 🛠️ Tecnologias Utilizadas
- 🐍 Python
- 🎯 Django
- 🗄️ SQLite
- 🎨 HTML & CSS
- 📦 Git & GitHub

## 🚀 Como Começar
1. Clone: `git clone https://github.com/Alvaro-Sousa/Exercicio-aula7-8.git`
2. Navegue: `cd Exercicio-aula7-8`
3. Instale: `pip install -r requirements.txt`
4. Migre: `python manage.py migrate`
5. Execute: `python manage.py runserver`
6. Acesse: `http://127.0.0.1:8000/`
