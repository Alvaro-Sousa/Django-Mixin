# Python Exercise Project using Django

## Description
This project is designed to help users learn and practice their skills in Python using the Django framework. It includes various exercises, showcasing the capabilities of Django and best practices in web development.

## Concepts
- Django framework fundamentals
- MVC architecture
- URL routing
- Database models and ORM
- Templates and static files
- Form handling and validation
- User authentication

## Technologies
- Python
- Django
- SQLite (or other relational databases)
- HTML & CSS
- Git & GitHub

## How to Run
1. Clone the repository:
   ```
   git clone https://github.com/Alvaro-Sousa/Exercicio-aula7-8.git
   ```
2. Navigate into the project directory:
   ```
   cd Exercicio-aula7-8
   ```
3. Install the required packages:
   ```
   pip install -r requirements.txt
   ```
4. Run migrations:
   ```
   python manage.py migrate
   ```
5. Start the development server:
   ```
   python manage.py runserver
   ```
6. Open a web browser and visit `http://127.0.0.1:8000/`.
